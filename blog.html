Question-3: What is event delegation in the DOM?

Ans:Event Delegation is a pattern based upon the concept of Event Bubbling. It is an event-handling pattern that allow to handle events at a higher level in the DOM tree other than the level where the event was first received.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling is a mechanism in the DOM (Document Object Model).when an event is triggered on an element, that event bubbles up through the DOM tree (from child to parent element).if a child element is clicked, that event will be triggered on its parent element.
